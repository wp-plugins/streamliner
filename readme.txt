=== Streamliner.co ===
Author: Chris Fong
Tags: streamliner, video, summary
Requires at least: 1.3

Inserts an embedded streamline from Streamliner (http://streamliner.co).

== Installation ==

1. Download [Streamliner plugin]
1. Unzip
1. Copy to your '/wp-content/plugins' directory
1. Activate plugin

Wordpress has instructions for installing plugins at [plugin installation page](http://codex.wordpress.org/Managing_Plugins)

== Usage ==

`[streamliner streamline_url width height]`

e.g.

`[streamliner http://srml.in/e93Cs/ 640 360]`

"Width" and "height" are optional parameters. The default if none is given: 640 pixels wide by 360 high.

You may use either a Streamliner "short" URL (available under "Shortlink" in the "More" menu on any Streamline page), or a full Streamline URL.
